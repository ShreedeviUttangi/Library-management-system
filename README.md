  # 📚 Nexus Grid

A modern and responsive **Library Management System** built using **HTML, CSS, JavaScript, React, TypeScript, and Vite**.  
The system includes separate portals for **Users, Employees, and Admins** with book management and analytics features.

---

# 🚀 Features

## 🏠 Home Page
- Display all available books
- Add new books
- Update existing books
- Delete books
- Responsive UI with modern design
- Red & Blue themed interface

---

# 👨‍🎓 User Portal
Students can:

- View books
- Search books
- Check book availability
- Explore latest books

---

# 👨‍💼 Employee Portal
Employees can:

- Add books
- Update book details
- Delete books
- Maintain library records

---

# 👨‍💻 Admin Portal
Admins can:

- Manage students
- Manage employees
- Monitor analytics
- Control system operations

---

# 📈 Advanced Features

## ✅ Last 5 Books Tracker
Tracks the latest 5 books added by employees.

## ✅ Monthly Popular Books Histogram
Displays the most popular books of the month using graphical analytics.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/library-management-system.git
```

## Navigate to Project

```bash
cd library-management-system
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```
